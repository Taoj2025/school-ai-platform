### 3.1 Apple 总览 (同学 5)

**时长**: 1 分钟

```

今天我将为大家演示 Apple 子系统的 4 大核心功能模块：
1. 奖状奖学金管理
2. 财务收支管理
3. 资产盘点管理
4. 学生事务管理

所有模块都支持主题切换，可根据学校需求定制界面风格。

让我带大家进入各个模块看看...
```

### 3.2 奖项管理 (同学 2)

**时长**: 5 分钟

#### 3.2.1 奖项总览 (1 分钟)

```
首先看奖项管理总览页面。

这里显示：
- 本学期奖项数量
- 待审核奖项
- 已发奖学金总额
- 历史累计

下方是奖项列表，支持搜索和筛选。
```

**操作**: 
- 展示统计数据
- 搜索"学业"

#### 3.2.2 新增奖项 (1 分钟)

```
点击新增奖项，填写表单：

- 奖项名称：学業優異獎
- 奖项类型：学业
- 学年：2025-2026
- 学期：上学期

保存后即可在列表中看到新奖项。
```

**操作**:
- 点击"新增奖项"按钮
- 填写表单
- 保存

#### 3.2.3 奖学金核算 (2 分钟)

```
点击奖项进入详情页。

左侧是获奖学生列表，显示学号、姓名、班级和成绩。

中间是奖学金核算面板，可以：
- 设置奖学金金额
- 自动计算总额
- 生成汇总表

右侧是 AI 确认区，OCR 提取的数据会在这里显示可信度。
```

**操作**:
- 进入奖项详情
- 展示学生列表
- 点击"计算奖学金"

#### 3.2.4 奖状生成 (1 分钟)

```
最后看奖状批量生成功能。

选择奖项后，可以：
- 多选获奖学生
- 预览奖状模板
- 一键生成 PDF
- 打包下载 ZIP

奖状会自动按班级、学号排序，方便颁奖典礼读稿。
```

**操作**:
- 进入"生成奖状"页面
- 选择学生
- 展示预览

### 3.3 财务 + 资产 (同学 3)

**时长**: 5 分钟

#### 3.3.1 财务管理 (2 分钟)

```
财务模块有 3 个标签页：

1. 收入记录：记录学费、捐赠、活动收费等
2. 支出记录：记录采购、维修、活动支出等
3. 统计报表：收入/支出分布图表

亮点功能：OCR 识别收据！
拍照上传，AI 自动提取金额、日期、付款人信息，
只需确认即可保存，大大减少手工输入。
```

**操作**:
- 展示收入 Tab
- 点击"OCR 识别"按钮
- 展示上传弹窗

#### 3.3.2 资产管理 (3 分钟)

```
资产模块同样有 3 个标签页：

1. 资产列表：查看和管理所有资产
2. 移动记录：追踪资产位置变更
3. 库存盘点：年度盘点任务管理

亮点功能：
- OCR 识别发票，自动填充资产信息
- 资产搬移记录，全程可追溯
- 一键生成盘点报告
```

**操作**:
- 展示资产列表
- 点击"OCR 识别"
- 点击搬移按钮，展示搬移弹窗

---

### 3.4 学生管理 + AI (同学 4)

**时长**: 5 分钟

#### 3.4.1 学生总览 (1 分钟)

```
学生管理模块是日常使用最频繁的功能。

顶部统计：
- 在读学生总数
- 本月考勤异常
- 待补领成绩表
- 在学证明待发

主表格显示所有学生信息，支持搜索。
```

**操作**:
- 展示统计卡片
- 搜索学生

#### 3.4.2 学生详情 (2 分钟)

```
点击学生进入详情页，有 4 个标签：

1. 基本信息：照片、学号、班级、家长联系
2. 考勤记录：出勤/迟到/缺席统计
3. 成绩记录：学期成绩（可扩展）
4. 证明申请：申请在学证明

所有信息一目了然，方便快速查询。
```

**操作**:
- 点击学生进入详情
- 展示基本信息 Tab

#### 3.4.3 批量导入 (1 分钟)

```
对于新学年批量导入学生：

1. 下载 Excel 模板
2. 填写学生信息
3. 上传文件
4. 预览导入结果
5. 确认导入

系统会自动检查重复和格式错误，
避免人工核对的大量工作。
```

**操作**:
- 点击"批量导入"
- 展示上传界面

#### 3.4.4 AI 证明生成 (1 分钟)

```
学生申请在学证明时：

1. 选择证明类型
2. 系统调用 AI 生成中英文内容
3. 用户确认信息
4. 一键生成 PDF

AI 会自动格式化内容，确保格式规范。
```

**操作**:
- 进入学生详情
- 点击"申请证明"

## 4. 常见问题

### Q1: 如何切换主题？
A: 点击右上角主题按钮，可选择浅色/深色/蓝色/绿色 4 种主题。

### Q2: OCR 识别失败怎么办？
A: 可以手动输入，系统会保留已识别内容供修改。

### Q3: 奖状模板可以自定义吗？
A: 可以，管理员可以上传自定义的 Word 模板。

### Q4: 数据如何备份？
A: 系统支持定期自动备份，也可手动导出 Excel。
