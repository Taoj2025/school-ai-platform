## 3. 端到端测试

### 3.1 测试场景 1: 奖项管理

**场景**: 管理员创建奖项并生成奖状

**步骤**:
1. ✅ 登录系统 (apple/admin)
2. ✅ 进入奖项管理页面
3. ✅ 点击"新增奖项"
4. ✅ 填写奖项信息并保存
5. ✅ 添加获奖学生
6. ✅ 计算奖学金
7. ✅ 生成奖状 PDF
8. ✅ 下载奖状

**结果**: ✅ 通过

### 3.2 测试场景 2: 财务记录

**场景**: 使用 OCR 识别收据并记录收入

**步骤**:
1. ✅ 进入财务管理页面
2. ✅ 点击"OCR 识别"
3. ✅ 上传收据图片
4. ✅ AI 识别内容
5. ✅ 确认并保存
6. ✅ 查看收入列表

**结果**: ✅ 通过

### 3.3 测试场景 3: 资产盘点

**场景**: 进行年度资产盘点

**步骤**:
1. ✅ 进入资产管理页面
2. ✅ 点击"开始新盘点"
3. ✅ 扫描资产标签
4. ✅ 记录资产状态
5. ✅ 生成盘点报告

**结果**: ✅ 通过

### 3.4 测试场景 4: 学生管理

**场景**: 批量导入学生并申请在学证明

**步骤**:
1. ✅ 进入学生管理页面
2. ✅ 点击"批量导入"
3. ✅ 上传 Excel 文件
4. ✅ 预览并确认
5. ✅ 选择学生申请证明
6. ✅ 生成证明 PDF

**结果**: ✅ 通过
